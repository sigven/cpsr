# Function that gets the maximum number of rows across different tier data frames in CPSR report

Function that gets the maximum number of rows across different tier data
frames in CPSR report

## Usage

``` r
get_max_rows_pr_datatable(cps_report)
```

## Arguments

- cps_report:

  CPSR report structure with tier data frames

## Value

max_row_nr maximum number of rows
