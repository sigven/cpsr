# Function that reads and validates an annotated germline SNV/InDel file from CPSR pre-reporting pipeline

Function that reads and validates an annotated germline SNV/InDel file
from CPSR pre-reporting pipeline

## Usage

``` r
load_germline_snv_indel(fname = NA, ref_data = NULL, settings = NULL)
```

## Arguments

- fname:

  Path to file name

- ref_data:

  Object with PCGR/CPSR reference data

- settings:

  Object with CPSR report configuration
