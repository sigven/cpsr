# Applies consistent styling using CPSR color palette.

Applies consistent styling using CPSR color palette.

## Usage

``` r
create_variant_table_theme(color_palette = NULL, header_color = NULL)
```

## Arguments

- color_palette:

  CPSR color_palette object

- header_color:

  Optional hex color for the header background. Defaults to
  color_palette\$report.

## Value

reactableTheme object
