# Cell renderer factory for genotype column Returns a function that renders genotype values as colored pills based on the provided color palette.

Cell renderer factory for genotype column Returns a function that
renders genotype values as colored pills based on the provided color
palette.

## Usage

``` r
rt_cell_genotype(color_palette)
```

## Arguments

- color_palette:

  CPSR color palette object containing genotype styling information

## Value

A function that takes a genotype value and returns an HTML span element
with appropriate styling
