# Cell renderer factory for classification rank column Maps a numeric rank (1=Benign ... 5=Pathogenic) to a labelled colored pill, enabling correct sort order via JavaScript.

Cell renderer factory for classification rank column Maps a numeric rank
(1=Benign ... 5=Pathogenic) to a labelled colored pill, enabling correct
sort order via JavaScript.

## Usage

``` r
rt_cell_classification_rank(color_palette)
```

## Arguments

- color_palette:

  CPSR color palette object

## Value

A function(value) returning an HTML span
