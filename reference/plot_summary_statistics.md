# Function that makes a piechart showing the number of variants at each significance level

Function that makes a piechart showing the number of variants at each
significance level

## Usage

``` r
plot_summary_statistics(variants_tsv, plot_type = "ClinVar")
```

## Arguments

- variants_tsv:

  data frame with variants in predisposition_genes

- plot_type:

  ClinVar or Other
