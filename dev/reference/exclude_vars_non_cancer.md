# Function that ignores/exclude variants attributed to non-cancer related phenotypes in ClinVar

Function that ignores/exclude variants attributed to non-cancer related
phenotypes in ClinVar

## Usage

``` r
exclude_vars_non_cancer(var_calls = NULL, conf = NULL)
```

## Arguments

- var_calls:

  variants in cancer predisposition genes

- conf:

  CPSR configuration object with run settings
