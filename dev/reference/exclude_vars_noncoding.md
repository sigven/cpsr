# Function that filters novel (non-ClinVar) variants based on a MAF threshold in gnomAD

Function that filters novel (non-ClinVar) variants based on a MAF
threshold in gnomAD

## Usage

``` r
exclude_vars_noncoding(var_calls = NULL, conf = NULL)
```

## Arguments

- var_calls:

  variants in cancer predisposition genes

- conf:

  CPSR configuration object with run settings
