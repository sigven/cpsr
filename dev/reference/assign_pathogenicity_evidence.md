# Function that assigns variant pathogenicity evidence based on ACMG guidelines

Function that assigns variant pathogenicity evidence based on ACMG
guidelines

## Usage

``` r
assign_pathogenicity_evidence(var_calls, settings, ref_data)
```

## Arguments

- var_calls:

  sample calls with dbnsfp annotations

- settings:

  cpsr settings object

- ref_data:

  pcgr data object

## Value

calls
