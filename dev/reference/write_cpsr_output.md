# Function that writes contents of CPSR report object to various output formats (quarto HTML reports, TSV, XLSX workbooks etc)

Function that writes contents of CPSR report object to various output
formats (quarto HTML reports, TSV, XLSX workbooks etc)

## Usage

``` r
write_cpsr_output(report, output_format = "html")
```

## Arguments

- report:

  List object with all report data (CPSR), settings etc.

- output_format:

  file format of output (html//tsv/xlsx etc)
