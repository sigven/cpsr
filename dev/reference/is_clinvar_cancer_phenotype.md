# Function that retrieves variants in cancer predisposition genes linked to cancer-related conditions according to ClinVar

Function that retrieves variants in cancer predisposition genes linked
to cancer-related conditions according to ClinVar

## Usage

``` r
is_clinvar_cancer_phenotype(cpg_calls, ref_data)
```

## Arguments

- cpg_calls:

  data frame with variants in predisposition_genes

- ref_data:

  object with PCGR/CPSR reference data
