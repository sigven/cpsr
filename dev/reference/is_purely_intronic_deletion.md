# Function that checks if a variant is a purely intronic deletion (i.e. deletion of intronic sequence that does not extend into exonic sequence at either end)

Function that checks if a variant is a purely intronic deletion (i.e.
deletion of intronic sequence that does not extend into exonic sequence
at either end)

## Usage

``` r
is_purely_intronic_deletion(hgvsc)
```

## Arguments

- hgvsc:

  HGVSc notation for variant

## Value

logical indicating whether variant is a purely intronic deletion
